# Først

**Først** er en minimal, friksjonsfri vane-app for å gjøre det viktigste først: kveldsbrief → fokusøkt → enkel oppgaveflyt.

## Hva den gjør
- **Kveldsbrief**: skriv morgendagens leveranse, tre kriterier og første steg.
- **I dag**: se leveranse + første steg, og jobb med **12 oppgaver** (MÅ/BØR/KAN). Hvis “MÅ” er gjort, er dagen vunnet.
- **Først-økt**: 90-minutters fokusøkt med *Parkering* (for distraksjoner) og *Resultat* (hva du produserte).
- **Flytt uferdige**: kopierer uferdige oppgaver til neste dag (uten å slette), med kapasitetssjekk.
- **Data**: importer/eksporter alt som JSON (uten innlogging).

## Prinsipper
- Ingen innlogging, ingen sync, ingen analytics.
- Lokalt lagret data med robust fallback.
- Bygget for vaner, ikke “produktivitetsstøy”.
